SKÅPA (or SKAPA, Skapa) is an app to create 3D printable boxes for IKEA SKADIS.

The models are generated in-browser using manifold-js and are rendered with ThreeJS using a custom shading pipeline.

This is a passion project by Nicolas Mattia. Read more in the release article: SKÅPA, a parametric 3D printing app like an IKEA manual.

The code is open-source and available on GitHub. You can support the project by giving it a star on GitHub and by reporting any issues you might encounter.
