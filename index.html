About

Usage

SKÅPA is a browser-based application that generates 3D-printable parts. It focuses on accessories for the IKEA SKÅDIS pegboard system. Specify dimensions and download the generated model. Slice with no supports and use 0.2 — 0.4 mm of layer height.

Implementation

SKÅPA is a client-side web application. Models are generated in-browser using manifold. Rendering is implemented with a custom Three.js effects pipeline. The app is built using the experimental Twrl framework. All source code is available on GitHub: nmattia/skapa.

Author

SKÅPA was created by Nicolas Mattia, a Swiss (not Swedish) software engineer. He works on Web graphics, developer tooling and embedded gadgets. He also writes on various topics, including on SKÅPA itself.
